## How to Run

1. **Temporarily relax Firestore rules** (or use the Firebase Emulator):
   ```
   rules_version = '2';
   service cloud.firestore {
     match /databases/{database}/documents {
       match /{document=**} {
         allow read, write: if true;
       }
     }
   }
   ```

2. **Call from any ViewModel or Activity** inside a coroutine scope:
   ```kotlin
   val seeder = SeedTestData(FirebaseAuth.getInstance(), FirebaseFirestore.getInstance())
   viewModelScope.launch {
       seeder.seedAll()
   }
   ```

3. **Restore Firestore rules** after seeding completes.

> **Note:** This is intended for one-time local/emulator testing only. Do not run on a production Firebase project with open rules.
